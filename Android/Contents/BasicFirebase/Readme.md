## FireBase의 형태 및 시작 / 구성
  ### 1. FireBase의 형태
  - 백엔드는 BaaS의 형태(로그인과 테이블 기능 제공)에서 PaaS 의 형태로 진화
  - Firebase는 PaaS(Platform as a Service)의 형태로 BaaS 보다 많은 기능들을 제공한다.
  - Firebase는 manifest에 직접 권한 설정을 하지 않아도 자동으로 처리해준다.
